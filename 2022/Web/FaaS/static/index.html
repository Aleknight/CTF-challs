Usage

Compute

Compute a result from an input. See compute_example.wasm (source) for an example. 

        
            $ curl http://.../compute -F input='{"a": 10, "b": 20}' -F code=@compute_example.wasm
            30

Compute on file

Compute a result from a file. See compute_file_example.wasm (source) for an example. alpaca.jpg is the demonstration file used in this example. 

        
            $ curl http://.../compute_file -F source=@alpaca.jpg -F code=@compute_file_example.wasm
            image/jpeg
